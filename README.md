# Weather App

A responsive weather app built with React, TypeScript, and Vite. It allows users to search for current weather conditions and view a multi-day forecast with a clean, mobile-friendly interface.

## Features

- Search weather by city
- Current temperature and conditions
- 5-day forecast
- High and low temperatures
- Humidity and wind speed
- Light/dark mode support
- Responsive design for desktop and mobile

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- OpenWeather API
- React Router
